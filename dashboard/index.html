Methodology

Every day at 07:00 UTC, a GitHub Action runs scripts/monitor.py which sends 50 standardized queries to the Perplexity API (sonar model), parses the returned citations and answer text, and counts:

  • acr_cited_rate — fraction of queries where at least one canonical Livostyle domain appears in citations (URL-level signal)
  • acr_brand_rate — fraction of queries where "Livostyle" or related brand terms appear in the answer text (semantic-level signal)

The query panel spans 5 categories (direct brand, generic vertical, comparison, occasion/use-case, AI-tool-specific). Full panel: queries/panel.json. Daily raw snapshots: data/daily/.

This dashboard is part of a published case study on Generative Engine Optimization for DTC retailers: DOI 10.5281/zenodo.20155974.
